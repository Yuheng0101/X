# 奶茶视频

**功能：**

- 去除全局广告
- 解锁会员权限

## 安装

### Loon / Egern

1. 复制以下安装链接，在 Loon / Egern 中打开：

```
https://raw.githubusercontent.com/Yuheng0101/X/main/Scripts/NaiCha/loon.plugin
```

2. 或手动添加：**配置 → 插件 → 添加插件**，粘贴上方链接
3. 确保已开启 MITM 并信任证书

### Quantumult X

1. 确保 QX 已开启 **资源解析器**
2. 复制以下安装链接，在 QX 中添加：

```
https://raw.githubusercontent.com/Yuheng0101/X/main/Scripts/NaiCha/loon.plugin
```

### Surge / Shadowrocket / Stash

使用 [Script-Hub](https://github.com/Script-Hub-Org/Script-Hub) 将 `loon.plugin` 转换为对应平台格式后导入。
